# 💳 Credit Card Fraud Detection System

Machine Learning based fraud detection system that predicts whether a credit card transaction is normal or fraudulent.

## Project Overview

This project uses Machine Learning to detect suspicious credit card transactions.

The model is trained using Random Forest Classifier and deployed using Streamlit.


## Features

- Transaction Amount analysis
- Transaction Hour analysis
- International Transaction detection
- Failed Attempts analysis
- Real-time fraud prediction


## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Random Forest
- Streamlit
- Google Colab


## Machine Learning Model

Algorithm:

Random Forest Classifier


## Application

Users can enter transaction details and the application predicts:

✅ Normal Transaction

or

🚨 Fraud Transaction


## Project Files

- app.py → Streamlit application
- fraud_model.pkl → trained ML model
- credit_card_fraud_synthetic.csv → dataset
